A Category object is the container object for a set of registry entries. Registry categories define sets of related or potentially related registry entries. For example, a registry category may be defined for equipment hierarchy level names (Enterprise, Site, Area, Work Center, Work Unit), which have alternate names on different systems. The combination of ID and CategorySourceID must be unique within a Registry.
